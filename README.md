# Roman Urdu Sentiment Analysis

## Overview

Roman Urdu Sentiment Analysis is a Natural Language Processing (NLP) project that classifies Roman Urdu text into sentiment categories such as Positive, Negative, and Neutral.

Roman Urdu is widely used on social media platforms, messaging applications, and online forums. Due to the absence of standardized spelling and grammar rules, sentiment analysis in Roman Urdu presents unique challenges. This project aims to address these challenges using machine learning techniques and text preprocessing methods.

## Features

- Text preprocessing and cleaning
- Roman Urdu sentiment classification
- Feature extraction using NLP techniques
- Machine Learning model training and evaluation
- Performance metrics and visualizations
- Easy-to-use and extensible codebase

## Dataset

The dataset contains Roman Urdu text samples labeled with sentiment classes.

Example:

| Text | Sentiment |
|--------|--------|
| Bohat acha product hai | Positive |
| Mujhe ye bilkul pasand nahi aya | Negative |
| Theek thaak tha | Neutral |
